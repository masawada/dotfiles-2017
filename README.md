dotfiles
========

## Install

Required:

- Bash
- Git
- curl
- Node.js

```console
$ bash -c "$(curl -fsSL https://dot.masawada.me/install)"
```

## Subcommands

- build - build dotfiles
- deploy - create symlink
- install - build && deploy
- clean - remove all symlinks and built files
- variables - show exported variables
